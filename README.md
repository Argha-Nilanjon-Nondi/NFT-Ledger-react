# NFT Ledger (Frontend)
### It is a frontend of nft ledger.

<br>

## Build with
- HTML5
- CSS3
- Javascript
- ES6
- ReactJS
- React-Router
- Axios
- Bootstrap 5
- React-Bootstrap

<br>

## Deployment

### To run the project

```bash
npm start
```

## Installation
### you need to setup this [NFT-Ledger-API](https://github.com/Argha-Nilanjon-Nondi/NFT-Ledger-api) repo as a backend in order to run this this repo
