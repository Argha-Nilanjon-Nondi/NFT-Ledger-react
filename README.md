# NFT Ledger (Frontend)
### It is a frontend of nft ledger.

<br>

## Build with
- HTML5
- CSS3
- Javascript
- ES6
- ReactJS
- React-Router
- Axios
- Bootstrap 5
- React-Bootstrap
